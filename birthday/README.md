# Birthday Surprise Website

A responsive, interactive birthday website ready for GitHub Pages.

## Personalize it

Open `script.js` and change the first few lines:

```js
const CONFIG = {
  name: "Beautiful Human",
  from: "Your favourite person",
};
```

You can also edit the wishes and reasons directly in `index.html`.

## Publish free with GitHub Pages

1. Create a new public repository on GitHub (for example, `birthday-surprise`).
2. Upload `index.html`, `styles.css`, and `script.js` to the repository.
3. Open the repository's **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`, then click **Save**.
6. After a minute or two, GitHub will show your live website link.

No installation or build step is needed.
